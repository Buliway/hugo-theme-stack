## Quickstart

```sh
hugo new site hugo-blog
cd hugo-blog
git clone https://github.com/Buliway/hugo-theme-stack/ themes/hugo-theme-stack
cp -r themes/hugo-theme-stack/exampleSite/* ./
rm hugo.toml
hugo server
```
